复习Linux文件权限

发表于 2026-04-10 分类于 技术笔记
Linux 文件权限的操作命令、目录与文件的权限区别

文件权限是什么

文件权限即当前文件对于文件所有者（user）、所属用户组（group）、其他人（other）而言所能执行的操作。
我们从文件信息来了解文件权限：

每一行的第一个字符表明文件的属性（d：目录；-：文件；l：链接文件）。
rwx 是可读、可写、可执行的意思。分为三组，依次规定 user\group\others 不同用户的权限。

权限修改命令

chgrp（change group ownership）：修改文件所属用户组。
chown（change file owner and group）：修改文件拥有者。
chmod（change file mode bits）: 修改文件的权限，

目录和文件的权限区别

文件：
r 是可读取文件内容；
w 是可以编辑文件内容，但不含删除文件；
x 是可以被系统执行的权限。

目录：
r 是可以查询该目录下的文件名数据；
w 是可以修改目录结构列表的权限（比如创建、删除、改名、移动文件或目录）；
x 是可以让该目录成为工作目录的权限。比方说，用户是否可以进入该目录的权限。
