源码安装支持python模块的vim

发表于 2026-04-29 分类于 技术笔记
通过源码安装让vim开启支持python、python3特性。

支持 python 的vim

make 环境

  • 安装链接库：sudo apt-get install libncurses5-dev libncursesw5-dev

来自 vim 源码安装文件夹中的 INSTALLx. txt 说明:
This document discusses cross-compiling VIM on Unix-like systems. We assume
You are already familiar with cross-compiling and have a working cross-compile
Environment with at least the following components:
a cross-compiler
a libc to link against
ncurses library to link against

  • 安装 make：apt-get install gcc automake autoconf libtool make

安装 vim

参考链接： https://vi.stackexchange.com/questions/11526/how-to-enable-python-feature-in-vim

  • 先下载 clone vim 的源码包：
  • git clone https://github.com/vim/vim.git
  • 解压它 tar -zxvf *** (如果下载后是一个文件夹，请忽略此步骤🙅‍)
  • 解压后的文件移动到/usr 下，别问为什么不直接解压到/usr 下（不知道，反正那个时候出现了 tab 会卡的情况！）
  • cd 到那个文件下的 src！
  • 配置、编译和安装：

参考链接： https://github.com/chxuan/vimplus/issues/130 （中文！）

[!info]
如果没有装 python-dev 和 python3-dev 的话，/usr/lib/python.. 那里会少了 config 文件夹

真正安装成功！
