Ubuntu默认不能root登录

发表于 2026-04-05 更新于 2026-04-08 分类于 技术笔记

Ubuntu 默认无法直接用 su root 登录 root 用户，原因是为了安全，默认给 root 设置了无密码状态，无密码则无法登录。
不过可以用 sudo -i/sudo su 来登录，它只需要输入当前用户的密码即可执行（对于安装系统时创建的用户是这样，这种用户默认在 sudo 组里边）。
