各科目学习板块

政治

对于政治，我的学习计划主要按三个板块展开：普通选择题、时政选择题、大题。
首先要明确的是，毛概和习概的内容主要是识记，从学习方法上看，也是以重复为主，而不像高数那样去理解知识点。
自己的学习体验下来，能讲明白毛概知识点的可能只有考研徐涛老师，专插本很多机构的课我都白嫖过（别找我要哈，我全删了 ~），都是以划知识点为主，但这对于专插本来说够用了。

  • 普通选择题
    • 资料：欢姐基础班、欢姐选择题、cb398 选择题
    • 具体实践：
      • 学习：看网课（推荐欢姐的网课，比较直接）在课本上划重点
      • 刷题：每天刷两个章节的选择题，一章是复习前一天刷过的，另一章是刷今天学习的。重点在于错误的选择题必须翻课本去复习，牢固知识点。可以把同一套题刷 n 遍。
    • 课本内的选择题，基本把这两套题刷熟了就能拿满分，偶尔可能会有一些特别细节的选择题这两套题没有涉及，但不用焦虑，因为通常情况下，特别细碎的知识点可能就一分。
  • 时政选择题
    • 资料：各个公众号都可以看看，不过他们给的实在太多了，想要精简些的话，推荐 cb398 精华笔记（最新的笔记通常都在次年的一二月份出版）
    • 具体实践：
      • 同样以刷题为主，但是这类选择题网上较少，可以尝试大声朗读来辅助记忆。亲测有效。
  • 大题
    • 资料：欢姐重点班的大题笔记，这份资料今年除了“文化繁荣兴盛”没踩中，其他都中了。
    • 具体实践：
      • 背背背！
      • 睡醒背，睡前背！
      • 考前必须把背过的全部复习一遍！我背了 144 道大题，其实背中了六道！但是其中两道考前没有复习，以至于考试的时候，记混淆了。巨后悔！

英语

本人高三第一次英语 35 分，高考英语 79 分。大二重新学英语，用的是《新概念英语》，学到第二册的时候，配合上刷题，英语四级低分飘过。基础薄弱的朋友可以参考我的方法。

  • 打基础
    • 网课：新概念英语 第二册哔哩哔哩bilibili
    • 具体实践：
      • 看课前先读两遍生词，记一下单词的意思
      • 边看网课边记知识点，不用记得太详细，学完一节课不需要超过 20 分钟
      • 认真完成 Summary writing、Key structures、Special difficulties
      • 跟读并达到可以熟读原文的程度。这个主要是培养语感，对学习语法大有脾益。
      • 方法参考自：第08讲 英语零基础 基础薄弱 | 怎么通过赖世雄美语从头学这套书打扎实基础哔哩哔哩bilibili
        • 《美语从头学》这本教材对我来说较之《新概念英语》没有那么有趣，故选择了后者。
  • 语法
    • 书籍：English Grammar in use
    • 我主要是拿这本书当工具书，遇到不懂的就拿出来查 ~ 如果专门花时间从头把这本书看到尾，很容易放弃的。
  • 刷真题
    • 刷题技巧我是直接照搬我四级的刷题技巧，这部分推荐大家看我是瑞斯拜，相同题型的做题技巧都是共通的，包括作文模板也是。

高数

  • 学习资料：
    • 课程学习：库课小红本；米哥专升本
    • 网课挑选方面，其实大家随意，宋浩老师的专插本课程、米哥专升本、杰哥专升本我都看过，没有打出差异性（个人认为），讲得都通俗易懂，大家挑一个适合自己的就行。
  • 刷题：
    • 我用的是库课的题海提升班，都是基础题，但是插本搞定基础题就完事了。
    • 不推荐库课必刷 2000，我高强度刷了一阵子（一天 6 页，同时还学别的）之后，人快 emo 了。里面很多题都不是插本会出的，大家刷几套真题卷就知道了。
    • 如果搞不到库课题海提升班资源的朋友，去刷各大公众号的题目，或者 b 站的代刷都是一样的，如果担心刷的题目“味儿”不正，那多刷真题感受一下，之后自行辨别即可。

C 语言

b 站很多编程大佬都说过，初学者学编程，建议先看机构的网课，后面学的编程语言，看文档或者书即可，我之前学过 Java，学 C 就直接看书了，非常高效。

  • 学习资料：
    • 官方指定教材——谭浩强教授著，C 语言程序设计（第四版）
    • 可能会有朋友说，这本书错漏百出，甚至配套的习题解析也有错误。事实确实如此，但试卷依然是按照教材来出的。
    • 如果大家在指针章节看得比较懵逼，可以去看《C Primer Plus》，里面对于指针的解释，非常简单易懂。
    • 对于变量赋值不明白的话，可以去看《C 和指针》第 79 页（人民邮电出版社）关于左值和右值的解释，朋友在问我 C 语言的问题时，我发现很多时候都跟不明白左值右值有关。
  • 具体实践：
    • 看书划知识点，不知道划哪些知识点很简单。每一章的小标题都很仔细，围绕小标题阅读，思考哪些跟小标题可以联系上，就是你要学习的知识点。
    • 做笔记：读完一章，我就会把这一章的小标题分级写在 markdown 文件里，然后合上课本，回忆这一章的知识点，用自己的话语描述出来，一次就够，知识点马上就能记熟。
    • 刷题：库课必刷 2000 题或库课小红本。
  • 简答题
    • 简答题说实话不是我的强项，我不大喜欢背诵，我考了 188分，估计 12分就丢在简答题。这东西就还是依靠理解来背诵吧，贝多芬 ~
    • 那么背哪些呢？看一下往年真题的简答题大家就能意会了。如下：
      • 二进制、十六进制（第二章：数据的存储运算）
      • 字符数组、输入输出的方法（第六章：数组）
      • 局部变量、全局变量（第七章：函数）
      • 指针、指针变量，它们的关系（第八章：指针）
      • 文件的组织形式有哪些（第十章：文件）
      • 数据、数据对象（第十一章：绪论）
      • 数据结构、数据类型（同上）
      • 串、子串（第十四章：串、数组和广义表）
      • 树的路径长度、带权路径长度（第十五章：树）
      • 二叉树、二叉树的五种基本形态（同上）
      • 算法、算法和程序的主要区别（第十九章：算法）
      • 递归是什么，它的优缺点是什么（第七章：函数 // 第十三章 栈和队列）

数据结构

数据结构直接看书就比较吃力，特别是新手直接看课本的伪代码那就更摸不着头脑了，推荐看王卓老师的数据结构课程：数据结构与算法基础（青岛大学-王卓）哔哩哔哩bilibili
刷题方面同 C 语言。
不过有的时候一些知识点看了王卓老师的解释也听不懂，放心！不是你的智商问题，只是授课方式可能刚好不适合你，这种时候去 b 站搜对应知识点，有些小 up 可能也是大学生，但做出来的视频却能讲好某个小知识点。

自控

有些人为什么天生就特别自律，不需要专门去寻找动力，就能专注学习，其实除去一部分环境因素，影响自律的最主要因素还是基因，也就是别人天生这方面就强。他们的基因决定了他们的前额叶皮质比别人发育得更快更好，更容易沉下心专注学习。不过天生这方面输给别人，也不是说就没救了，了解点脑科学可以给你提供点帮助。
如果为了自律去了解一些脑科学，或者说是为了学习锻炼专注力、培养习惯的方法，而去看了一些科普书，学习之余充电放松，我觉得也蛮赚的，这方面的书籍我看过几本，推荐《自控力》、《原子习惯》和《贪婪的多巴胺》。
《自控力》会让你知道，专注力这东西用起来就跟肌肉一样，会累，也会变强。
《原子习惯》则是一堆非常实用的原则，帮助你培养好习惯。
《贪婪的多巴胺》用很多例子解释了多巴胺不会让你快乐，而是让你产生动力等机制。
不过懒得去看这几本书的朋友，可以试试我比较常用的几个方法：

  • 冥想锻炼专注力这块“肌肉”。很简单，闭上眼睛感受鼻子呼吸，新手从 3 分钟开始，然后 5、8、12、30 分钟慢慢锻炼；
  • 简化早起流程，避免学习前消耗过多精力。可以参考帅 soserious 的这个视频：【每日一遍】5个起床习惯帮你充满动力！哔哩哔哩bilibili。可以根据自己的状况调整，我状态最好的一段时间是起床之后，上个厕所就开始学习。
  • 大家可能都听说过：给自己制定清晰简明的任务可以避免拖延。我根据这个原则优化了一下：提前把各个学科的学习内容和学习方式列成清单，如果哪天我的计划被打破，或者说提前完成了当天计划，不知道要学什么的时候，直接拿起清单选一个即可。
